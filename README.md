## ✨ 개발 배경

다양한 갱년기 증상으로 인해 중년기 삶의 질 저하됩니다.
홀링은 동반자의 갱년기 증상상태를 확인하고 갱년기에 대한 이해도를 높여, 함께 갱년기와 중년기의 삶을 함께 걷는 호르몬 힐링 서비스입니다.
